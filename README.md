## 🧠 内容覆盖

仓库中的代码主要涉及以下算法与数据结构方向：

- 基础算法：排序、枚举、模拟、前缀和、差分、二分、双指针
- 数据结构：栈、队列、链表、堆、哈希表、并查集、树状数组、线段树
- 图论算法：DFS、BFS、最短路、最小生成树、拓扑排序、连通性问题
- 动态规划：线性 DP、背包 DP、区间 DP、树形 DP、状态压缩 DP
- 数学算法：快速幂、欧几里得算法、组合数学、质数筛、取模运算
- 字符串算法：KMP、字符串哈希、字典树等
- 竞赛技巧：贪心、构造、分治、剪枝、对拍与随机数据生成

## 🚀 如何运行

本仓库代码大多为单文件 C++ 程序，可使用 `g++` 进行编译运行。

普通编译：

```bash
g++ main.cpp -o main
./main
```

使用 C++17 标准：

```bash
g++ main.cpp -std=c++17 -O2 -Wall -o main
./main
```

Windows 环境下也可以运行：

```bash
g++ main.cpp -std=c++17 -O2 -Wall -o main.exe
main.exe
```

仓库中包含部分对拍与数据生成代码，用于在算法竞赛训练中验证程序正确性。

这部分内容主要用于提升代码调试能力和竞赛稳定性。
