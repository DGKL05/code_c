问题背景与挑战

问题描述

无人机需要飞过一系列障碍物，每个障碍物i有高度范围约束[li, ri]。

初始高度h0=0，高度变化由数组d决定，每个di只能是0或1，但可能存在未知值(-1)需要确定。

目标：确定所有未知di的值，使无人机路径满足所有障碍物的高度约束，或判断不可能。

核心挑战

组合爆炸问题

若有k个未知值，直接枚举需2^k种可能，当k较大时不可行

动态范围维护

如何高效维护每一步的可行高度范围，确保约束条件满足

双向协同处理

正向与逆向处理需协同工作，才能高效确定未知值

正向范围维护算法

核心思想

维护每一步的可行高度范围[curr_low, curr_high]，根据当前高度变化（已知或未知）计算下一步范围，并与障碍物约束求交集。

处理逻辑

初始化当前高度范围: curr_low = curr_high = 0
for 每个障碍物i in 1..n:
获取障碍物约束 [li, ri]
if di已知(0或1):
new_low = curr_low + di
new_high = curr_high + di
else (di=-1):
new_low = curr_low + 0
new_high = curr_high + 1
// 与障碍物约束求交集
curr_low = max(new_low, li)
curr_high = min(new_high, ri)
if curr_low > curr_high: 无解，返回-1
结束循环
