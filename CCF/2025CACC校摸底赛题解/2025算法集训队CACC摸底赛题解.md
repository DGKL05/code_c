# A 交换最小值和最大值

## 解题思路

注意一下，如果要先交换最小值与arr[0]，要判断一下max_arr 是否等于arr[0]，如果要先交换最大值与arr[n-1]，要判断一下min_arr是否等于arr[n-1]，至于如何实现两数交换这些是基础内容，直接调用库函数即可，当然如果你进行函数封装手动实现的话，要注意进行地址传参。（可以参考AC Code中的sw_ap函数）。

## AC Code

```cpp
#include<bits/stdc++.h>
using namespace std;
using ll =long long;

void sw_ap(int &a,int &b){
    int temp = b;
    b = a;
    a = temp;
    // cout<<"a = "<<a<<" b = "<<b<<"\n";
}
void test()
{
    int n=0;
    cin>>n;
    int min_idx=-1;
    int max_idx=-1;
    int min_arr=1000000;
    int max_arr=-1;
    vector<int>arr(n,0);
    for(int i=0;i<n;i++){
        cin>>arr[i];
        if(arr[i] < min_arr){
            min_arr = arr[i];
            min_idx = i;
        }
        if(arr[i] > max_arr){
            max_arr = arr[i];
            max_idx = i;
        }
    }
    // cout<<min_idx<<" "<<max_idx<<"\n";
    swap(arr[0],arr[min_idx]);
    if(max_idx == 0) max_idx = min_idx;
    swap(arr[n-1],arr[max_idx]);
    for(int i=0;i<n;i++){
        cout<<arr[i]<<" ";
    }
}

int main()
{
    std::ios_base::sync_with_stdio(false);
    int T=1;
    // cin>>T;
    while(T--) test(); 
    return 0;
}

```

# C 猴子吃桃问题

## 题解思路

每天先吃一半的桃子，再额外吃一个桃子。那么第N天是1个桃子的话，那么前一天 也就是第N-1天就是 (1+1)*2.。也就是说第i天的桃子数量  等于（第i+1天（后一天）的桃子数量 + 1）* 2。

题目N给的范围很小，直接for循环模拟即可。

## AC Code

```cpp
#include<bits/stdc++.h>

{
    int n=0;
    cin>>n;

        ans*=2;

    cout<<ans<<"\n";
}

int main()
{
    std::ios_base::sync_with_stdio(false);
```

## D **强迫症**

## 解题思路（1）（[前置知识 前缀和]([https://](https://oi-wiki.org/basic/prefix-sum/)https://)）

手办的高度最多为10000，不卡空间。开一个10005大小的数组，其中下标表示处于该高度的手办数量，也就是说arr[i]为10的话代表有10个手办的高度为i。然后进行前缀和处理，也就是arr[i+1]=arr[i] + arr[i+1]。此时，arr[i]就表示，手办高度小于等于i的手办一共有多少个。那么arr[i+k] - arr[i-1]就表示 手办高度在[i,i+k]的区间的手办数量，每次枚举i取arr[i+k] - arr[i-1]的最大值，即可。

## 解题思路（2）

有没有办法，不需要任何的前置算法知识呢？有的 有的xd。

我们会发现，高度在[i,i+k]区间，与高度在[i+1,i+1+k]的区间，这俩个区间的手办数量有什么区别呢？很明显，高度在[i+1,i+1+k]的手办数量，比前者，少了高度为i的手办数量，多了高度为i+1+k的手办数量。

那么高度在[i+1,i+1+k]的区间，与高度在[i+2,i+2+k]这俩个区间呢，根据上面的内容，可以发现，也就是少了，高度为i+1的手办数量，多了高度为i+2+k的手办数量。

那么我们如何实现以上内容，我们依然开一个10005大小的数组，其中下标表示处于该高度的手办数量也就是说arr[i]为10的话代表有10个手办的高度为i。先计算一遍[0,k]这个区间的手办数量，然后定义俩个指针，一个是头（st），指向0，一个是尾（en），指向k。每次更新，先减去arr[st]，在让指针后移，st++;en++;，然后在加上arr[en]，就是当前这个高度区间的手办数量。代码大家自己尝试实现一下，注意一下数组越界的问题就好了。

## AC Code（思路 1 ）

```cpp
#include<bits/stdc++.h>
using namespace std;
using ll =long long;

void test()
{
    int n=0;int k=0;
    cin>>n>>k;
    vector<int>sum(10005,0);
    for(int i=0;i<n;i++){
        int num=0;
        cin>>num;
        sum[num]++;
    }
    for(int i=1;i<=10000;i++){
        sum[i]+=sum[i-1];
    }
    int ans=0;
    for(int i=1;i<=10000;i++){
        if((i+k)<=10000){
            ans = max(ans , sum[i+k] - sum[i-1]);
        }
    }
    cout<<ans<<"\n";
}

int main()
{
    std::ios_base::sync_with_stdio(false);
    int T=1;
    cin>>T;
    while(T--) test(); 
    return 0;
}
```

# E h0359. 并查集

## 悄悄话

出这题的想法就是看看，有没有队员默写不出并查集的板子，当然也是看看有没有新生已经学到并查集了的。

## 解题思路（[前置知识 并查集](https://oi-wiki.org/ds/dsu/)）

直接套用以下模版即可

```cpp
struct DSU {
    int n;
    vector<int>pre;
    DSU (int n){
        pre.resize(n+1);
        for(int i=0;i<=n;i++) pre[i] = i;
    }
    int find(int x){
        return pre[x] == x ? x : find(pre[x]);
    }
    void unid (int a,int b){
        a = find(a);
        b = find(b);
        if(a == b) return ;
        pre[a] = b;
    }
    bool check(int a,int b){
        return find(a) == find(b) ? true : false;
    }
};
```

# F 深入虎穴（前置知识 [DFS](https://oi-wiki.org/graph/dfs/) and [BFS](https://oi-wiki.org/graph/bfs/)）

## 解题思路

题目中三个需要注意的点

1. 并没有规定入口是1号。
2. 007 发现不存在两条路通向同一扇门。
3. 迷宫只有一个入口。

因为不确定几号是入口

1. 如果你遍历每个点当做入口，应该是会TLE的（ps：虽然我没交这样的代码进行测试）。
2. 如果你直接把1号当入口，那么最多也就只有18分。

我们分析一下条件二 ： 007 发现不存在两条路通向同一扇门。

这说明这个图，其实是颗特别的树，树有一个特点就是，根节点没有父亲（也就是没有点能到达根节点，只存在根节点到达其他结点）。再结合第三个条件 ： 迷宫只有一个入口。那么根节点就是我们要找的入口。要找根节点就简单很多了，只要有点不能达到，那么这个点就是根节点，并且这个点只会有一个。我们只需要开数组标记结点是否可抵达即可。

确定了入口，要找最远结点就很简单了，利用BFS（广度优先搜索的特点）就可以找到，最后一个出队的点就是，我们要找的答案。

## AC Code

### 存图的方式有很多种，包括本题找最远点，也不一定要用BFS，即使是BFS也可以不用我这样定义队列

```cpp
#include<bits/stdc++.h>
using namespace std;
using ll =long long;

vector<vector<int>>arr;

struct node{
    int idx;
    int pre;
};

int ans;

void bfs(int st){

    queue<node>q1;
    q1.push({st,st});

    while(q1.size()){
        auto t = q1.front();
        q1.pop();
        ans = t.idx;
        for(auto next : arr[t.idx]){
            if(next == t.pre) continue;
            q1.push({next,t.idx});
        }
    }
}
void test()
{
    int n=0;
    cin>>n;
    arr.resize(n);
    vector<bool>fa(n);
    for(int i=0;i<n;i++){
        int m=0;
        cin>>m;
        for(int j=0;j<m;j++){
            int a=0;
            cin>>a;
            a--;
            fa[a] = true;
            arr[i].push_back(a);
        }
    }
    int st=-1;
    while(fa[++st]);
    bfs(st);

    cout<<ans+1<<"\n";
}

int main()
{
    std::ios_base::sync_with_stdio(false);
    int T=1;
    // cin>>T;
    while(T--) test(); 
    return 0;
}
```
